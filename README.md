# Berlin and Monaco Public Map Selector

Static hosting bundle for the Berlin and Monaco modified maps.

## Maps

- Full Open: `maps/full-open/index.html?mode=training`
- Full Barriers: `maps/full-barriers/index.html?mode=training`
- Full Closed: `maps/full-closed/index.html?mode=training`

## Monaco maps

- Full Open: `maps/monaco-full-open/index.html?mode=training`
- Full Barriers: `maps/monaco-full-barriers/index.html?mode=training`
- Full Closed: `maps/monaco-full-closed/index.html?mode=training`

Open `index.html` through any static web server. Unity WebGL files should be
served over HTTP/HTTPS, not directly from `file://`.
